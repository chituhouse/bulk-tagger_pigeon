# Changelog

## 0.8.9 – 2025-07-04

### Added
- Manifest 驱动工作流，导出阶段统一生成 TXT。
- 批量重新生成功能，支持多选图片一次性获取新 Prompt。
- 预览面板文件名高亮显示。
- 全选 / 部分选 / 取消选 三态复选框逻辑。

### Fixed
- 列表自定义高亮导致黑色背景问题 → 使用 Qt 内置选中样式。
- 操作按钮需重复选图才能生效的问题。
- 线程清理与错误提示，防止 UI 卡死。

### Removed
- 旧版自动即时写 TXT 的实现。
- 多个调试脚本与临时文档已归档到 `docs/archive/`。
